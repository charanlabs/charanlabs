# Hi, I'm Charan

I build **DevOps platforms, cloud automation, and AI-assisted engineering workflows** that help teams ship faster with stronger reliability, security, and developer experience.

My focus areas:

- **Platform Engineering**: self-service platforms, golden paths, and developer enablement
- **DevOps & Cloud**: CI/CD, Infrastructure as Code, Kubernetes, GitOps, and cloud automation
- **Agentic DevOps**: AI-assisted PR review, infrastructure change analysis, incident support, and operational workflows
- **Reliability & Security**: observability, policy-as-code, guardrails, and production readiness

## What I'm Building

- **Internal Developer Platforms** that reduce cognitive load for engineering teams
- **Golden Path Templates** for secure, reusable service delivery
- **GitOps and IaC Workflows** for scalable and auditable infrastructure changes
- **Agentic DevOps Systems** with human approvals, traceability, and safe automation
- **Observability Foundations** with SLOs, dashboards, alerting, and incident learnings

## Engineering Principles

- Build platforms like products
- Prefer self-service over ticket-driven operations
- Automate with guardrails, not with blind trust
- Make reliability and security part of the delivery path
- Optimize for developer experience and operational clarity
